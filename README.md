# Halil Işık - Kişisel Portfolyo Web Sitesi 🚀

Bilgisayar Mühendisliği öğrencisi Halil Işık'ın çalışmalarını, yeteneklerini ve projelerini sergilediği modern, premium ve responsive (mobil uyumlu) one-page kişisel portfolyo web sitesi.

Sitenin canlı önizlemesini ve tasarımını temsil eden görsel özellikleri aşağıda listelenmiştir.

## 🎨 Tasarım Özellikleri & Teknolojiler
- **Core Stack:** HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **Tema & Estetik:** Gece-mavisi ve mor gradyan temalı, modern **Glassmorphism** kart tasarımları.
- **Mikro Etkileşimler:**
  - Giriş alanında dinamik yazar kasa efekti (Typewriter animation).
  - Proje kartlarında fare hareketini izleyen özel **imleç parlaması (Cursor Glow)**.
  - Yumuşak kaydırma (smooth scroll) ve aktif bölüm takibi (Intersection Observer ile Scrollspy).
- **İletişim:** Web3Forms API entegrasyonu ile Cloudflare korumalı, çalışan canlı iletişim formu.
- **Performans:** CDN veya harici kütüphane bağımlılığı olmaksızın, tamamen yerel inline SVG ikon altyapısı sayesinde yüksek sayfa yüklenme hızı.

## 📂 Proje Yapısı
```bash
├── index.html     # Sayfa yapısı ve içerikler
├── style.css      # CSS değişkenleri, animasyonlar ve responsive tasarım sistemimiz
├── script.js      # Etkileşimler, animasyon kontrolleri ve form gönderimi
└── README.md      # Proje tanıtım belgesi
```

## 🛠️ Yerel Çalıştırma
Projeyi kendi makinenizde çalıştırmak için dosyaları indirip `index.html` dosyasını tarayıcınızda açmanız yeterlidir. VS Code kullanıyorsanız **Live Server** eklentisi ile `http://127.0.0.1:5500` adresinde yerel sunucu başlatabilirsiniz.
